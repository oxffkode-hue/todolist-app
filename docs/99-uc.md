# TodoListApp Use Case Diagram

## 전체 유스케이스 다이어그램

```mermaid
graph LR
    %% 액터 정의
    Guest(["👤 비회원"])
    User(["👤 인증된 사용자"])

    %% 시스템 경계
    subgraph System["TodoListApp 시스템"]

        subgraph Auth["인증 및 계정 관리"]
            UC01(["UC-01\n회원가입"])
            UC02(["UC-02\n로그인 / 로그아웃"])
            UC03(["UC-03\n개인정보 수정"])
            UC09(["UC-09\n회원 탈퇴"])
        end

        subgraph TodoMgmt["할일 관리"]
            UC04(["UC-04\n할일 등록"])
            UC05(["UC-05\n할일 목록 조회"])
            UC06(["UC-06\n할일 수정 및 완료 처리"])
            UC07(["UC-07\n할일 삭제"])
        end

        subgraph CategoryMgmt["카테고리 관리"]
            UC08(["UC-08\n카테고리 관리"])
            DC["기본 카테고리\n(업무 / 개인 / 기타)"]
        end

        subgraph Filter["필터링"]
            F1["카테고리 필터"]
            F2["기간 필터\n(dueDate 기준)"]
            F3["완료 여부 필터"]
        end

    end

    %% 비회원 관계
    Guest -->|"가입 요청"| UC01
    Guest -->|"로그인"| UC02

    %% 인증된 사용자 관계
    User -->|"로그아웃"| UC02
    User --> UC03
    User --> UC04
    User --> UC05
    User --> UC06
    User --> UC07
    User --> UC08
    User --> UC09

    %% include / extend 관계
    UC04 -.->|"«include»"| UC08
    UC05 -.->|"«extend»"| F1
    UC05 -.->|"«extend»"| F2
    UC05 -.->|"«extend»"| F3
    UC08 -.->|"«include»"| DC

    %% 스타일
    style Guest fill:#E8F4FD,stroke:#2196F3,color:#000
    style User fill:#E8F5E9,stroke:#4CAF50,color:#000
    style System fill:#FAFAFA,stroke:#9E9E9E
    style Auth fill:#FFF3E0,stroke:#FF9800
    style TodoMgmt fill:#F3E5F5,stroke:#9C27B0
    style CategoryMgmt fill:#E0F2F1,stroke:#009688
    style Filter fill:#FCE4EC,stroke:#E91E63
    style DC fill:#ECEFF1,stroke:#607D8B,color:#000
```

---

## 액터 정의

| 액터 | 설명 |
|------|------|
| 비회원 (Guest) | 시스템에 미가입 또는 미인증 상태의 사용자 |
| 인증된 사용자 (Authenticated User) | 회원가입 후 JWT 토큰으로 인증된 사용자 |

## 유스케이스 요약

| UC | 명칭 | 주요 액터 | 관련 BR |
|----|------|----------|---------|
| UC-01 | 회원가입 | 비회원 | BR-01, BR-02 |
| UC-02 | 로그인 / 로그아웃 | 비회원, 인증된 사용자 | BR-03 |
| UC-03 | 개인정보 수정 | 인증된 사용자 | BR-02 |
| UC-04 | 할일 등록 | 인증된 사용자 | BR-03, BR-04, BR-05 |
| UC-05 | 할일 목록 조회 및 필터링 | 인증된 사용자 | BR-06, BR-07, BR-12, BR-13 |
| UC-06 | 할일 수정 및 완료 처리 | 인증된 사용자 | BR-04, BR-05, BR-06 |
| UC-07 | 할일 삭제 | 인증된 사용자 | BR-06, BR-07 |
| UC-08 | 카테고리 관리 | 인증된 사용자 | BR-08, BR-09, BR-10, BR-11 |
| UC-09 | 회원 탈퇴 | 인증된 사용자 | BR-02, BR-03, BR-14 |

---

## 관계 설명

| 관계 | 설명 |
|------|------|
| UC-04 «include» UC-08 | 할일 등록 시 카테고리 선택이 필수로 포함됨 |
| UC-05 «extend» 필터링 | 할일 목록 조회에서 카테고리/기간/완료 여부 필터를 선택적으로 적용 가능 |
| UC-08 «include» 기본 카테고리 | 카테고리 관리 시 기본 카테고리(업무/개인/기타)는 항상 제공되며 수정/삭제 불가 |
